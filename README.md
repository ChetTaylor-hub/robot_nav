# Yolov5_ros

This package provides a ROS wrapper for [PyTorch-YOLOv5](https://github.com/ultralytics/yolov5) based on PyTorch-YOLOv5.. The package has been tested with Ubuntu 16.04 and Ubuntu 18.04.
